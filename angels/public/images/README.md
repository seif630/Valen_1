# Product Images

This folder contains product images for the Angel's e-commerce website.

## How to Add Images:

1. **Upload via Admin Panel:**
   - Go to `/admin` in your browser
   - Select and upload your product images
   - Use the generated URLs in your database

2. **Manual Upload:**
   - Add image files directly to this folder
   - Name them descriptively (e.g., `white-tshirt.jpg`, `blue-jeans.jpg`)
   - Reference them in database as `/images/filename.jpg`

3. **Sample Images:**
   For testing, you can:
   - Download free stock photos from Unsplash
   - Use placeholder services like Lorem Picsum
   - Create your own product photos

## Image Specifications:

- **Format:** JPG, PNG, WebP
- **Size:** Max 5MB per image
- **Dimensions:** Recommended 800x1200px for product images
- **Naming:** Use descriptive names (e.g., `red-dress-front.jpg`)

## Current Sample Data:

The database is configured to look for these images:
- `/images/white-tshirt.jpg`
- `/images/blue-jeans.jpg`
- `/images/summer-dress.jpg`
- etc.

Add these files or update the database with your actual image URLs.
